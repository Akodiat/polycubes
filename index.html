What is a polycube?

"A polycube is a solid figure formed by joining one or more equal cubes face to face" - Wikipedia

Ok, but what is this model about?

An output polucybe is generated by stochastically assembling it from an input set of building block cubes. Each cube can have up to six patches (one at each side) and the patches will bind toghether if they have complementary colours and the same orientation.

So what do I do?

Change the input by adding, removing, or modifying the available cube types. You can change the patch colours by changing the corresponding numbers. Positive numbers will bind to negative: 1 to -1, 2 to -2, etc. An empty patch has the value 0 and will not bind. You can also change the patch orientations by clicking the button.
